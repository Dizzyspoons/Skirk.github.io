How the Bot Works

1

Command System

Skirk Bot responds to commands with either slash commands (/) or the double dash prefix (--). Both work the same way, so use whichever you prefer!

/hunt or --hunt
2

Custom Prefix

Server admins can change the command prefix to something custom using the /setprefix command. This allows for multiple bots with different prefixes in the same server.

/setprefix !
3

Check Prefix

Forgot what prefix is being used in your server? Simply use the /prefix command to see the current active prefix for Skirk Bot.

/prefix
4

Admin Controls

Server admins have additional control over the bot, including enabling/disabling it in specific channels and managing announcement settings.

/enable #channel

How to Play

Getting Started Guide

  1. Invite Skirk Bot to your Discord server using the button above
  2. Use /apply to register as a hunter
  3. Start hunting monsters with /hunt to collect loot
  4. Sell your loot for Mora using /sell
  5. Use Mora to buy Primogems in the shop with /shop
  6. Pull characters and weapons with /wish
  7. Build your team and battle others with /team and /battle

Privacy & Data Policy

Skirk Bot is designed with your privacy and data safety in mind. We only collect the bare minimum required to provide an immersive and enjoyable experience. We do not track personal activity, and your data is never sold or shared.

✅ What We Collect

  • Your Discord user ID
  • Game-related data (wishes, team setup, inventory, levels)
  • Command usage (e.g. `--hunt`, `/profile`, `/pvp`)

❌ What We Don't Collect

  • Message logs or chat history
  • Presence data logs (no online/offline storage)
  • Personal identifiable info (email, IP, etc.)
  • Anything unrelated to gameplay

🔐 How We Use Your Data

  • To enable core features like profiles, gacha pulls, and leaderboards
  • To personalize your experience (team builds, wallpaper settings)
  • To improve game balance and engagement over time

🔄 Data Retention & Security

  • Your data is stored securely on encrypted servers
  • Game data is retained for 365 days of inactivity
  • You can request data deletion at any time through our support server
  • We implement industry-standard security measures to protect your information

🔄 Opt-Out & Removal

If you want your data deleted, just contact us through our support server. You can also disable presence-based features by going invisible or not using commands that rely on online status.
